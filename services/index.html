Data Analysis and Visualization

These services are meant for utilities or consultants facing a specific problem who already have a well-defined and accesible dataset they wish to analyze.

Examples:

  • Identification of areas of concern for PFAS (or any contaminant) based on water quality sample data

Sample deliverables:

  • 3-5 static or dynamic visualizations along with annotated analysis and discussion of results
  • One follow-up analysis included in initial scope-of-work
  • Opportunity to follow-up with more in-depth analyses

Dashboarding

Dashboarding services are meant for utilities or consultants looking for insight into their data on-demand. Unlike a static analysis, dashboards are designed to deal with data that may change over time.

Examples:

  • A real-time dashboard that displays disinfectant residual values in the distribution system based on samples processed in an external software tool.

Sample deliverables:

  • 3-5 static or dynamic visualizations in dashboard format
  • Provisioning: any number of users can be granted access to the dashboard by creating a login and password
  • Accesibility: dashboard is available 24/7 on-demand
  • Real-Time: dashboard can be designed to update automatically. Requires a connection to a data source

Artificial Intelligence and Machine Learning

AI and ML services involve anything around prediction, anomaly detection, or classification, and are reserved for more advanced problems.

Examples:

  • Predict whether a residential water meter is malfunctioning
  • Predict whether a water treatment plant is experiencing a contamination event

Sample deliverables:

  • A model is delivered as Python code. Note that to use the model, a User Interface and possibly a backend database architecture may be required as well.

Full-Stack Web Development

Building an app? These services are for utilites, consultants, or startups who are looking for software engineering expertise.

Example:

  • Building an API integration to connect data from two different software systems
  • Designing and building the front-end of a webpage using HTML, CSS, and JavaScript frameworks such as React.js

Digital Product Management

Product Management services are for those looking for guidance on how to go about understanding the needs of their end users, and how to design, build, and deploy your product to maximize your chances of success.

Example:
